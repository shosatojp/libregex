# libregex
Regular expression library for C

## Available features
* `+` `*` `.` `^` `$` `(...)` `(?:...)` `(...|...)` `[...]` `[^...]` `?` `+?` `*?` `{m, n}` `{m}` `\t` `\r` `\n` `\f` `\d` `\D` `\s` `\S` `\w` `\W` `\escape` `0-9`

* capture
* find all by regex


## Build

```sh
./autogen.sh
./configure
make
```
